# syntax
Go syntax consist of the following:
* 1. Package declaration
* 2. Import packages
* 3. Functions
* 4. Statements and expressions

## package declaration
package declaration tells go that the program belongs to the package

## import packages
it import files included in the fmt package. while fmt is used to print messages in our terminal

## Functions
it holds the whole workflow of the program

## Statements and expressions
In Go, statements are separated by ending a line (hitting the Enter key) or by a semicolon ";".
